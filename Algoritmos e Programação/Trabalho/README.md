## Sistema AmazonCC

Você foi contratado(a) para criar um sistema que gerencie as operações de uma loja virtual
utilizando a linguagem Python. Seu programa deve realizar operações semelhantes a uma
empresa de venda de produtos virtuais.
O seu programa deve permitir que os clientes façam o cadastro (1), insiram seus itens em um
carrinho de compras (2), vejam a lista de itens adquiridos (3) e finalizem a compra (4).

### Executar
```python3 sistema_amazonCC.py```
