Sobre

Shrek é um ogro que vivia feliz e sozinho num pântano, em meio à floresta, em uma terra chamada Duloc. Repentinamente, ele vê sua solidão ameaçada quando o governante de Duloc, Lord Farquaad, decide expulsar todas as criaturas mágicas para floresta.
